# Vecindar

**Tu barrio conectado**

Sistema integral de gestión comunitaria para barrios privados.

## Features

- 🚨 Botón de pánico (SOS)
- 🗺️ Mapa de incidentes con heatmap
- 📋 Sistema de requerimientos/reclamos
- 🐕 Mascotas perdidas
- 👥 Gestión de vecinos y lotes
- 📊 Dashboard de administración

## Stack

- **Frontend:** Next.js 15 + React + Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** PostgreSQL + Prisma
- **Maps:** Leaflet + OpenStreetMap
- **Hosting:** Vercel + Supabase

## Desarrollo

\`\`\`bash
npm install
npm run dev
